Менеджер файлов
Программа управления файлами написанная на Qt, которая читает файл с диска в память и выводит информацию о нем. 

Затем, другой командой, файл можно сохранить под другим именем на диске.
Т.е. есть две опции - Прочитать файл..., Сохранить файл как...

Файлы могут быть большим, поэтому имеется уведомление пользователя о статусе загрузки.

Уведомление выглядит как:
Прогресс бар с процентами, как во всех файловых менеджерах.
Отображает среднюю скорость записи и чтения.
Отображает мгновенную скорость и предполагаемую продолжительность операции.

Окно приложения должно корректно масштабироваться.

Дополнительно:
1. Имеется пауза текущей операции с возможностью продолжения.
2. Пишется лог событий.
